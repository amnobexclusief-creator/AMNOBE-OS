# Visie

Doel: Beschrijf kort de missie van AMNOBE OS, doelgroep en succescriteria.

- Missie: Een modulair AI-platform dat organisaties helpt workflows te automatiseren.
- Doelgroep: Freelancers, kleine bedrijven, interne productteams.
- Succescriteria: eerste betalende klant binnen 3 maanden; stabiele MVP; reproduceerbare deployment.

## Kernprincipes
- Modulaire architectuur
- Memory als centrale dienst
- Test + documentatie per feature
